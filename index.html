Honours Project - Procedural Generation of Cities

Research into how procedural generation can enhance game production. For more info on my project you can click on the picture to view my proposal.

Unreal

My honours project is in the early stages of development. I chose this project as I have never done procedural generation before and I was always interested in how it works. The plan for this project is to generate a simple city with a road network to compare and analyse to other cities in games. I will compare how realistic it is compared to other cities in games and its effectiveness as a city in different genres of games. The project will be determined a success if the quantitive data from the survey is positive. I will also gather performance data to ensure the tool is efficient. So far I am on track with my project plan and development has been smooth.

  • Game Engine: Unreal Engine 4
  • Programming Language: C++
  • Platform: PC

Beano - Fun Force

The professional project module consisted of different disciplinaries to form a team and create a minimum viable product for a client. We got given a brief set by Beano studios. They client wanted us to create a mobile game that is inspired from marioware and the mario party games. We chose to follow the same path of mario party and create and implement a game with an overworld, a series of minigames and a score system. We developed our game in Unity and we took on a minigame each and some of the menu/overworld mechanics. My minigame was a concept inspired by angry birds where the player would throw pies at the enemies going past the screen. This project gave us a real understanding of how a development company should operate and how to set realistic deadlines. Our client worked very closely with us which was a great help in driving the game in the direction they wanted.

  • Game Engine: Unity
  • Programming Language: C#
  • Platform: Mobile
